# AI Interview Platform

An AI-powered interview preparation platform built using the MERN Stack.

## Features Implemented

* User Registration
* User Login
* JWT Authentication
* Protected Routes
* Get Current User API
* Create Interview API
* MongoDB Integration
* Password Hashing with bcrypt

## Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcryptjs

## API Endpoints

### User Routes

POST /api/users/register

POST /api/users/login

GET /api/users/me

### Interview Routes

POST /api/interviews
